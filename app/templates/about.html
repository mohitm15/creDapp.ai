About

CreÐApp.ai is the first ever decentralized AI-driven credit rating solution platform which replaces the middlemen using an automated AI_driven, distributed and transparent ledger. We call it an Intelligent Contract.

The user fills a loan registration form which consists of various generic parameters such as Income Amount, Credit Amount, Gender, Etc. After filling the form, the data is passed to a decentralized storage system made on Ethereum Ropsten TestNet. This data now cannot be meddled with and goes to the flask server for AI computation. The AI model predicts if the user is viable for the loan or not and sends back a credit score. This is also sent through the smart contract and back to the user.
